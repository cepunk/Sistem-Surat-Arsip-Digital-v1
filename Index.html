Hubungkan Spreadsheet Kustom

Anda dapat memindahkan penyimpanan data aplikasi ini ke Google Sheet lain dengan menempelkan ID file di bawah.

Cara Mendapatkan ID Spreadsheet Anda

1. Buka file Google Spreadsheet yang ingin Anda hubungkan di browser.

2. Perhatikan bagian kolom alamat URL browser Anda. Format URLnya adalah:

https://docs.google.com/spreadsheets/d/ID_SPREADSHEET_ANDA/edit#gid=0

3. Salin blok kode panjang di antara huruf /d/ dan /edit tersebut, lalu tempelkan ke form input di atas.

Penting: Akun Google yang meng-host Web App ini wajib memiliki akses "Editor" pada Spreadsheet target agar sinkronisasi CRUD tidak error.
